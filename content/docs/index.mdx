# PayCrypt API Documentation

PayCrypt offers a comprehensive API for accepting cryptocurrency payments, managing wallets, and automating subscriptions. This documentation covers the complete API reference with examples to help you integrate PayCrypt into your application.

## Getting Started

To use the PayCrypt API, you'll need:

1. A PayCrypt account
2. API credentials (access keys)
3. Basic understanding of RESTful APIs and cryptocurrency transactions

## Authentication

All API requests must include your API key in the `Authorization` header:

```bash
Authorization: Bearer YOUR_API_KEY
```
